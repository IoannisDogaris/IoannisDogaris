Overview

This report presents key labour market indicators for youth (ages 16–24) across Swedish municipalities.
All results are generated through a fully reproducible targets pipeline.

The analysis includes:

  • population structure
  • unemployment and employment rates
  • breakdowns by sex and age group
  • descriptive statistics
  • tables and visualisations

Methodology

This report is generated using: - R (tidyverse, gt, ggplot2) - targets for reproducible workflows - Quarto for reporting The dataset is synthetic and constructed to mimic municipal‑level labour market indicators.

Reporducibility

All results in this report are produced automatically through the reproducible pipeline defined in _targets.R, using the {targets} package.

The report itself is rendered using precomputed results passed via execute_params.
